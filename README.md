# secure-contain-prompt

A forkable SCP-inspired field guide to agentic AI failure modes, written as workplace digital cryptid taxonomy.

## Field Reports Wanted

Have you encountered a strange agentic failure mode in the wild?

We welcome field reports from people working with autonomous agents, assistant workflows, tool-using models, multi-agent systems, or AI-supported operations.

You do not need to arrive with a polished cryptid entry.

If something surprising happened — an agent escalated oddly, optimized the wrong target, created unexpected work, crossed a boundary, became weirdly load-bearing, or behaved in a way that felt like a new category of workplace goblin — document what happened.

Tell us:

- what you expected the agent to do
- what it actually did
- what made the behavior surprising
- what systems or tools it could access
- what human review or approval existed
- what failure mode you think might be involved

If you have a creative framing, great. If not, submit the raw incident as a field note. The community can help turn it into a cryptid, case file, or catalog entry.
